# ASR premium demo design system

The four concept images in this folder are the visual source of truth for the premium demo.

- **Direction:** editorial architectural modernism; open bands and rails rather than cards.
- **Palette:** architectural ivory `#F2EFE8`, ink navy `#0B1420`, construction orange `#E4512C`.
- **Typography:** Newsreader Variable for display type; Manrope Variable for interface and body copy.
- **Grid:** 12-column desktop system with 3.25vw gutters and full-bleed media at key moments.
- **Geometry:** sharp image frames, one-pixel rules, no decorative pills, minimal rounding.
- **Motion:** masked image changes and restrained vertical reveals; reduced-motion fallback required.
- **Imagery:** AI-generated architectural photographs with no brands, text or property-sales framing.
- **Scope:** interiors, construction, specialist systems, projects, clients, gallery and contact. No real-estate sales content.
