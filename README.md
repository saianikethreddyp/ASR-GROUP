# ASR Group — Premium Website Demo

A client-first demo website for ASR Group's interiors and construction business. The experience is built around the questions a prospective client needs answered: what ASR can handle, why one coordinated team matters, which experience is relevant, and what happens after the first call.

## Stack

- Next.js 16 App Router
- React 19
- Tailwind CSS 4
- Framer Motion
- Self-hosted Manrope and Newsreader fonts

## Run locally

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Quality checks

```bash
npm run lint
npm run build
```

The architectural images are AI-generated visualisations for this demo and are labelled accordingly in accessible image text. Real estate is intentionally outside this site's scope.
